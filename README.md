# Sales Analysis Dashboard

This project analyzes global sales data using Python, Pandas, and Matplotlib. It includes visualizations of sales trends, category-wise sales, top countries and regions, sales distribution, and sales vs quantity plots.

---

## Project Files

- `main.py` – Main Python script for data analysis and generating plots  
- `sales.csv` – Dataset containing sales data  

---

## Key Features

- Visualizes sales trend over time with median highlights  
- Category-wise sales comparison  
- Top countries and regions analysis  
- Sales distribution histogram  
- Sales vs quantity scatter plot  

---

## Technologies & Dependencies

- Python 3.x  
- Pandas  
- Matplotlib  
- python-dateutil  

### Install Dependencies

```bash
pip install pandas matplotlib python-dateutil
```
## Insights
-Sales increased significantly after the first quarter of 2013
-Technology category contributed most to overall sales
-United States leads in total sales
-Central region has the highest share of sales
-Majority of sales are in the 0–500 USD range
-High-value sales (over 1000 USD) are rare


## Usage

1. Clone this repository:
```  
git clone https://github.com/amanml2026/sales-analysis-dashboard.git
```
2.Navigate to the project folder:
```
cd.sales-analysis-dashboard
```
3.Run the analysis script:
```
python main.py

```
